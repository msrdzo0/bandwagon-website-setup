## 什么是搬瓦工 VPS 建站绑定域名

先说清楚概念，后面操作起来不会迷。

搬瓦工（BandwagonHost）VPS 建站绑定域名，指的是把你注册的域名（比如 example.com）通过 DNS 解析记录，指向你搬瓦工 VPS 服务器的 IP 地址，之后访客输入域名就能打开你跑在这台服务器上的网站。域名本身只是一个"名字"，服务器的 IP 才是"地址"，DNS 解析就是这两者之间的翻译层。

搬瓦工本身不提供域名解析服务——你需要借助域名注册商自带的 DNS，或者用 Cloudflare、DNSPod 这类第三方解析平台来做这件事。

## 第二步：配置域名 DNS 解析，把域名指向搬瓦工 IP

这一步有两种主流路线：**用 Cloudflare** 或者**用 DNSPod**。两个都免费，操作差不太多。国内用户如果不确定选哪个，Cloudflare 自带免费 CDN，网站被攻击时可以开启保护，多一层防御；DNSPod 操作界面是中文，更直观。

### 方案 A：用 Cloudflare 解析（推荐）

1. 注册并登录 Cloudflare，点 "Add a Site"，输入你的域名
2. Cloudflare 会自动扫描域名当前的解析记录，一般直接点 Continue
3. 选择免费套餐（Free），点 Continue
4. 手动添加 A 记录：
   - Type：A
   - Name：`@`（代表根域名 example.com）
   - IPv4 address：填入你搬瓦工的 IP 地址
   - 是否开启代理（小云朵）：如果 VPS 线路本身够快，建议先关掉（灰色）
5. 再添加一条 www 的 A 记录，Name 填 `www`，其他一样
6. Cloudflare 会给你两个 NS 服务器地址，格式类似 `xxx.ns.cloudflare.com`
7. 去你的域名注册商后台，找到 Name Server 或 DNS 服务器设置，把默认的 NS 改成 Cloudflare 给你的那两个
8. 等待生效，快的话几分钟，慢的话几个小时

### 方案 B：用 DNSPod 解析

1. 登录 DNSPod，添加你的域名
2. 添加两条 A 记录：主机记录分别填 `@` 和 `www`，记录值都填搬瓦工的 IP，TTL 用默认 600 就行
3. DNSPod 会给你 NS 地址，同样去域名注册商处修改 Name Server 指向 DNSPod 的 NS

### 验证解析是否生效

等解析生效后（通常 5 分钟到几小时），用 `ping 你的域名` 命令，或者在网上找个 DNS 查询工具，输入你的域名查 A 记录，看返回的 IP 是否是你搬瓦工的 IP。对上了就说明解析成功。

不生效的话先别急，等等看。要是超过 24 小时还不行，大概率是 Name Server 没改对，回去再检查一遍。

## 第三步：在搬瓦工 VPS 上搭建网站环境

域名解析配好之后，才轮到在服务器上装网站环境。顺序别搞反了——先解析再装环境，能实时看到域名访问效果。

目前搬瓦工建站最常用的三种方案：

### 方案一：宝塔面板（最适合新手）

宝塔面板是带图形界面的服务器管理工具，Nginx/Apache/MySQL/PHP 一键安装，建站、绑定域名、申请 SSL 证书都能在网页里操作，不用记命令行。小内存 VPS（1GB）跑起来也没太大问题。

安装命令（Ubuntu/Debian 系统）：

bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh


装好之后，宝塔会给你一个面板地址和初始账号密码，登进去添加站点，填入你的域名，选 PHP 版本，数据库同时建好，就基本配置完了。

### 方案二：LNMP 一键包（性能好一点）

LNMP = Linux + Nginx + MySQL + PHP。Nginx 对小内存服务器更友好，1GB 内存跑 WordPress 没问题。适合不怕命令行的用户。

安装步骤：

1. SSH 连接到搬瓦工 VPS
2. 运行 LNMP 安装包脚本，安装过程中选好 MySQL 和 PHP 版本
3. 安装完成后，用 `lnmp vhost add` 命令添加一个虚拟主机，这一步就是绑定域名——把你解析好的域名填进去
4. 添加完会提示是否设置伪静态，WordPress 选 wordpress 规则

### 方案三：手动安装 Nginx + PHP + MySQL

完全自定义，适合有一定 Linux 基础的用户。在 Nginx 的 server 块里配置 `server_name 你的域名`，重启 Nginx，域名就和网站目录绑上了。

说实话对大多数建博客、做个人站点的人来说，宝塔面板是最省事的选择，用它就行，没必要花时间折腾手动安装。

## 第四步：安装 WordPress，网站正式上线

服务器环境装好、域名也解析到位之后，装 WordPress 就是最后一步。

**用宝塔安装 WordPress 的流程：**

1. 宝塔面板 → 软件商店 → 一键部署 → 找到 WordPress，填入你的域名，数据库信息自动填充
2. 部署完成后，浏览器访问你的域名，进入 WordPress 安装向导
3. 填写站点标题、管理员账号、密码、邮箱，点"安装 WordPress"
4. 安装完成，后台地址是 `你的域名/wp-admin`

**用 LNMP 手动安装：**

1. 下载 WordPress 压缩包，通过 SFTP 或 `wget` 命令上传到 `/home/wwwroot/你的域名/` 目录
2. 解压，浏览器访问域名，跟着安装向导走一遍，填数据库信息（数据库地址填 localhost）

安装向导里有一步设置伪静态，WordPress 要把固定链接设置开，不然文章页面打开是 404。进入 WordPress 后台 → 设置 → 固定链接，选一个你喜欢的格式（推荐"文章名"），保存。

## 搬瓦工建站绑定域名常见问题

**Q：搬瓦工建站需要域名备案吗？**

不需要。搬瓦工的服务器在境外，境外 VPS 建站不需要 ICP 备案，直接用域名访问就行。这也是很多人选择搬瓦工建站的原因之一——省去了备案的等待时间和流程。

**Q：域名解析后多久能访问网站？**

DNS 解析生效通常需要几分钟到几十分钟，极少数情况下会到 24 小时。如果超过 1 小时还不行，先用 `ping 你的域名` 确认 IP 是否指向正确，再检查 Name Server 有没有改成功。本地 DNS 缓存也会影响——试试换个网络或者用手机流量访问。

**Q：一台搬瓦工 VPS 能不能绑定多个域名、建多个网站？**

可以。宝塔面板支持添加多个站点，每个站点绑定不同域名，各自独立运行。LNMP 也支持多虚拟主机。唯一要注意的是 VPS 内存和硬盘够不够分配——1GB 内存跑 2-3 个轻量级站点没问题，再多就可能出现 MySQL 频繁挂掉的情况。

**Q：SSL 证书怎么装，域名怎么变成 https？**

宝塔面板内置了 Let's Encrypt 免费 SSL 证书申请功能，在站点设置里点"SSL"，填好域名，一键申请，全程自动，证书有效期 90 天自动续期。LNMP 也有对应的 certbot 操作方式。装好之后 https 就能用了，建议同时设置 http 强制跳转 https，对 SEO 也有好处。

**Q：搬瓦工支持退款吗，买错了怎么办？**
